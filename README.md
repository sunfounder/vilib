## Install

Download this repository to your Raspberry Pi:

```bash
cd ~
git clone -b picamera2 https://github.com/sunfounder/vilib.git --depth 1
```

Install

```bash
cd ~/vilib
sudo python3 install.py 
```

## Usage

```bash
cd ~/vilib/examples
sudo python3 xxx.py
```

Stop running the example by using <kbd>Ctrl</kbd>+<kbd>C</kbd>
